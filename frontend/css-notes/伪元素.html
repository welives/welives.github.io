选择器 描述 CSS 版本
::after 在匹配元素之后插入内容 2
::before 这匹配元素之前插入内容 2
::first-letter 匹配元素的第一行第一个字母 1
::first-line 匹配元素的第一行 1
::selection 匹配元素中被用户选中或处于高亮状态的部分 3

# ::before

我是原本就有的内容
<div class="test-before">我是原本就有的内容</div>
<style>
  .test-before::before {
    content: '我是通过::before插入的内容';
    background-color: #e2e2e2;
    font-size: 20px;
    height: 30px;
    padding: 5px 10px;
    border: 1px solid red;
  }
</style>

# ::after

我是原本就有的内容
<div class="test-after">我是原本就有的内容</div>
<style>
  .test-after::after {
    content: '我是通过::after插入的内容';
    background-color: #e2e2e2;
    font-size: 20px;
    height: 30px;
    padding: 5px 10px;
    border: 1px solid red;
  }
</style>

::before和::after 能做的事不止是插入文本内容，还能插入图片、icon 图标等
