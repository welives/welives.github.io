选择器 描述 CSS 版本
:link 匹配所有未访问链接 1
:visited 匹配所有访问过的链接 1
:hover 匹配把鼠标放在元素上的状态 1
:active 匹配被用户激活的元素 1
:checked 匹配所有选中的表单元素 3
:disabled 匹配所有被禁用的元素(主要是表单元素) 3
:enabled 匹配所有启用的元素(主要是表单元素) 3
:empty 匹配所有没有子元素的元素 3
:focus 匹配获得焦点的元素 2
:focus-within 匹配任意元素或其子元素获得焦点 3
:first-child 匹配一组兄弟元素中的第一个元素 2
:last-child 匹配一组兄弟元素中的最后一个子元素 3
:only-child 匹配没有任何兄弟元素的元素 3
:first-of-type 匹配一组兄弟元素中，某种类型的元素的第一个元素 3
:last-of-type 匹配一组兄弟元素中，某种类型的元素的最后一个元素 3
:only-of-type 匹配任意一个元素，这个元素没有其他相同类型的兄弟元素 3
:nth-child(n) 匹配一组兄弟元素中的第 n 个子元素 3
:nth-last-child(n) 匹配一组兄弟元素中的倒数第 n 个子元素 3
:nth-of-type(n) 匹配一组兄弟元素中，某种类型的元素的第 n 个元素 3
:nth-last-of-type(n) 匹配一组兄弟元素中，某种类型的元素的倒数第 n 个元素 3
:not(selector) 匹配不符合一组选择器的元素 3
:root 匹配文档的根元素。:root表示<html>元素，除了优先级更高之外，与 html 选择器相同 3
:target 匹配 id 值与当前的活动锚点相同的元素 3
:in-range 匹配所有可输入表单元素中，其值在指定范围内的元素 3
:out-of-range 匹配所有可输入表单元素中，其值在指定范围外的元素 3
:valid 匹配所有表单元素中，内容验证正确的元素 3
:invalid 匹配所有表单元素中，内容未通过验证的元素 3
:required 匹配所有设置了"required"属性的元素 3
:optional 匹配所有未设置"required"属性的可输入表单元素
:read-only 匹配所有不可编辑的元素 3
:read-write 匹配所有可以编辑的元素 3
:indeterminate 匹配状态不确定的表单元素 3
:lang(language) 匹配"lang"属性是指定起始值的元素 2

# :nth-child

这个 CSS 伪类首先找到所有当前元素的兄弟元素，然后按照位置先后顺序从 1 开始排序，选择的结果为 nth-child（an+b） 括号中表达式匹配到的元素集合

  • :nth-child(n)或:nth-child(1n)或:nth-child(1n+0)
    • 匹配全部
  • :nth-child(2n+1)或:nth-child(odd)
    • 匹配奇数项
  • :nth-child(2n)或:nth-child(even)
    • 匹配偶数项
  • :nth-child(0n+1)或:nth-child(1)
    • 匹配第一个，等价于:first-child
  • :nth-child(-n+b)
    • 匹配前 b 项
  • :nth-child(n+b)
    • 从第 b 项开始往后匹配(包括 b)
Strong 1! Small 2 Em 3! B 4 I 5! Span 6 Span 7!
查看代码 
<div class="nth-child">
  <strong>Strong 1!</strong>
  <small>Small 2</small>
  <em>Em 3!</em>
  <b>B 4</b>
  <i>I 5!</i>
  <span>Span 6</span>
  <span>Span 7!</span>
</div>
<style>
  .nth-child :nth-child(n + 3) {
    color: red;
  }
</style>

# :nth-last-child

  • :nth-last-child(0n+1)或:nth-last-child(1)

# :indeterminate

  • <input type="checkbox">元素，其 indeterminate 属性被 JavaScript 设置为 true 时处于不确定状态
  • <input type="radio">元素, 表单中拥有相同 name 值的所有单选按钮都未被选中时处于不确定状态
  • 处于不确定状态的<progress>元素

男女
查看代码 
<div class="indeterminate">
  <input type="checkbox" />
  <label>被设置为不确定状态时是绿色的</label>
  <br />
  男<input type="radio" name="sex" />女<input type="radio" name="sex" />
  <label>未选择时应该是绿色的</label>
  <br />
  <progress></progress>
</div>
<style>
  .indeterminate :indeterminate + label {
    background-color: rgba(0, 255, 0, 0.25);
  }
  .indeterminate :indeterminate {
    opacity: 0.5;
    box-shadow: 0 0 2px 1px red;
  }
</style>
<script>
  let inputs = document.querySelectorAll('.indeterminate input')
  let progress = document.querySelector('progress')
  inputs.forEach((item) => {
    item.indeterminate = true
  })
  setTimeout(() => {
    progress.max = 100
    progress.value = 80
  }, 5000)
</script>
